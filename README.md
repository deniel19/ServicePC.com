# 🖥️ PC Repair Service Website

A clean, modern, single-page website for a PC repair and support service — built with **HTML, CSS, and JavaScript**. Features an automatic image slider, service list, customer reviews, and a simple assistant input box.

---

## 📸 Features

- ✅ **Image Slider** – Cycles through 3 PC build images every 5 seconds with smooth fade transitions
- ✅ **Text-Based Logo** – No image logo required (clean and professional)
- ✅ **Services List** – Clearly displays offered services with checkmarks
- ✅ **Customer Reviews** – Shows ratings and testimonials
- ✅ **Assistant Input** – Simple text box with Enter key alert
- ✅ **Responsive Design** – Works on desktop and mobile
- ✅ **Single File** – All code in one `index.html` file — no external CSS or JS needed
